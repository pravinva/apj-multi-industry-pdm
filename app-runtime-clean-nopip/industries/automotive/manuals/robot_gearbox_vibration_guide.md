# Robot Gearbox Vibration Service Guide (Excerpt)

Equipment Class: Body shop welding robot axis gearbox assemblies
Revision: D5.1

## Failure Mode Guidance
- Failure mode: gearbox_wear
- Typical indicators: vibration_rms rise, harmonic_peak at gear mesh frequency, axis_repeatability drift
- Confirmation criteria: vibration_rms > 9.0 mm/s for 15 minutes and repeatability error > 0.35 mm

## Inspection and Repair
- Inspection interval: every 140 operating hours
- Torque: 55 N-m on gearbox mounting bolts
- Part number: GBX-RBT-A7
- Part number: LUBE-SYN-220

## Parts and Readiness
- Reorder point guideline: 2 gearbox kits per robot cell cluster
- Lead time for GBX-RBT-A7: 14 days
- If risk-adjusted demand exceeds on-hand by >1 kit, raise expedited requisition

## Prescriptive Action Notes
- For high anomaly with rising harmonic peak, plan intervention within 24h
- Run post-repair vibration baseline for 30 minutes and store baseline signature
- Record replaced lot and technician notes to support model feedback coverage
