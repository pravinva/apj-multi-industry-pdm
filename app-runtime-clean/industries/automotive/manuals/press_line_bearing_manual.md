# Press Line Bearing Maintenance Manual (Excerpt)

Equipment Class: Press line servo drive and spindle assemblies
Revision: A3.2

## Failure Mode Guidance
- Failure mode: bearing_wear
- Typical indicators: vibration_rms increase, temperature drift, torque ripple
- Confirmation criteria: vibration RMS > 8.5 mm/s for 30 minutes, temperature_limit: 95 C

## Inspection and Repair
- Inspection interval: every 168 operating hours
- Torque: 42 N·m on bearing housing cap bolts
- Part number: BRG-6208-2RS
- Part number: SEAL-35x52x7

## Parts and Readiness
- Minimum on-hand recommendation: 2 sets per press line
- Lead time for BRG-6208-2RS: 9 days
- If stock below reorder point, expedite purchase request within same shift

## Prescriptive Action Notes
- If anomaly score >= 0.8 and RUL < 24h: plan intervention in current shift.
- If anomaly score between 0.5 and 0.8: schedule intervention in next 24-48h.
- Record action outcome in CMMS for model feedback and retraining.
