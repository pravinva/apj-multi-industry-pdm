# Inverter Cooling Loop Reliability Manual (Excerpt)

Equipment Class: EV inverter cooling skid, pump, plate heat exchanger
Revision: B2.4

## Failure Mode Guidance
- Failure mode: coolant_flow_restriction
- Typical indicators: inlet_outlet_delta_t rise, pump_current increase, pressure_drop oscillation
- Confirmation criteria: pressure_drop > 18 kPa for 20 minutes and coolant_flow < 75 percent nominal

## Inspection and Repair
- Inspection interval: every 96 operating hours
- Torque: 16 N-m on manifold clamp bolts
- Part number: FLT-INV-450
- Part number: PMP-SEAL-44A

## Parts and Readiness
- Minimum on-hand recommendation: 3 filter kits per line
- Lead time for FLT-INV-450: 7 days
- If two or more skids trend above restriction threshold, pull forward procurement by one week

## Prescriptive Action Notes
- If anomaly score >= 0.78 and RUL < 36h, schedule intervention in the next planned downtime window
- Validate coolant conductivity before restart
- Capture post-maintenance thermal profile and attach to CMMS ticket
